# Ecommerce-Api
It is an ecommerce api build with node, express, typescript


## Requirements:
### Language: 
   Typescript (Node)
### Framework: 
   Express
### Database:
   MongoDB
### Libraries:
   - **ODM**: Mongoose
   - **Validation**: Joi
   - **Authentication and Authorization**: Json Web Tokens
   - **Hashing**: Bcrypt
